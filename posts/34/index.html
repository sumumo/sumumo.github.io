Windows 11 右键菜单添加 Git Bash

发表于 2023-08-25 阅读次数：

个人感觉 Windows 11 的终端挺好用的，在终端中使用 Git Bash 也舒服多了，想要在某个文件夹中直接打开 Git Bash 的时候需要按住 Shift + 右键打开原来的 Git Bash，下面提供两种方法可以直接右键在终端中打开 Git Bash 到指定目录。

安装 Git 的时候不要忘记勾选下面的选项：

方法一：设置终端默认配置文件为 Git Bash

在终端设置中选择默认配置文件为 Git Bash，如果没有 Git Bash 选项的话需要自行添加，之后直接在文件夹中点右键选择 “在终端中打开” 即可。

方法二：使用软件添加自定义右键菜单

先安装 Context Menu For Windows 11，安装完成后打开软件，添加一条新的菜单（需要终端中有 Git Bash 的配置文件，没有的话要自己添加）：
